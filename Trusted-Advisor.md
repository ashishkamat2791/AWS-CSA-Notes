# Trusted Advisor

Automatic Advisor which checks your system for possible improvment

Typical checks:
    - Service limits
    - Security Groups with unrestricted access
    - IAM use
    - MFA on Root Account

Four areas convered:
- Security
- Performance
- Cost Optimization
- Fault Tollerance 

Red - Action recommended
Yellow - Investigation recomended
Green - all good

You can exclude ceratin checks
